# AdMob Banner + Interstitial Ads Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Google AdMob banner and interstitial ads to Aria, visible only to non-Premium users, configured via `local.properties` locally and GitHub Actions secrets in CI.

**Architecture:** A pure `InterstitialCadence` class decides *when* to fire an interstitial (interaction count + cooldown), independent of the Android/AdMob SDK. A Hilt-singleton `AdManager` wraps the real AdMob SDK calls (init, load, show) and uses `InterstitialCadence` internally. `MainViewModel` calls `AdManager.recordInteraction()` on every user turn and exposes `AdManager.showInterstitialEvents` for `MainScreen` to collect and trigger the actual full-screen ad. A new `BannerAdView` composable renders the banner above the input bar, gated on `!isPremium`.

**Tech Stack:** Kotlin, Jetpack Compose, Hilt, `com.google.android.gms:play-services-ads`, JUnit + Mockito-Kotlin (existing test stack), Gradle Kotlin DSL.

## Global Constraints

- AdMob App ID: `ca-app-pub-8382831211800454~4907127144`
- Banner ad unit ID: `ca-app-pub-8382831211800454/8429916670`
- Interstitial ad unit ID: `ca-app-pub-8382831211800454/2093261545`
- Ads must be completely hidden for Premium users (`FeatureGate`/`BillingManager.isPremium == true`).
- `local.properties` must contain the real IDs for local builds and must **never** be committed (already gitignored — do not change `.gitignore` handling).
- CI (`.github/workflows/build.yml`) must source the same three values from GitHub Actions repository secrets: `ADMOB_APP_ID`, `ADMOB_BANNER_ID`, `ADMOB_INTERSTITIAL_ID`.
- Follow the existing `readProp()` pattern in `app/build.gradle.kts` (used for `github.api.token` etc.) rather than inventing a new config mechanism.
- Interstitial cadence: every 4th user interaction, minimum 60-second cooldown between shows.
- Banner placement: bottom of `MainScreen`, directly above the floating input bar.

```kotlin
package com.aria.assistant.ads

class InterstitialCadence(
    private val interactionsPerInterstitial: Int = 4,
    private val cooldownMillis: Long = 60_000L,
    private val clock: () -> Long = { System.currentTimeMillis() }
) {
    private var interactionCount = 0
    private var lastShownAtMillis = -cooldownMillis

    fun recordInteraction(): Boolean {
        interactionCount++
        val dueByCount = interactionCount % interactionsPerInterstitial == 0
        val cooldownElapsed = clock() - lastShownAtMillis >= cooldownMillis
        return dueByCount && cooldownElapsed
    }

    fun markShown() {
        lastShownAtMillis = clock()
    }
}
```

```kotlin
package com.aria.assistant.ads

import android.app.Activity
import android.content.Context
import com.aria.assistant.BuildConfig
import com.aria.assistant.billing.BillingManager
import com.aria.assistant.engine.AriaLogger
import com.google.android.gms.ads.AdError
import com.google.android.gms.ads.AdRequest
import com.google.android.gms.ads.FullScreenContentCallback
import com.google.android.gms.ads.LoadAdError
import com.google.android.gms.ads.MobileAds
import com.google.android.gms.ads.interstitial.InterstitialAd
import com.google.android.gms.ads.interstitial.InterstitialAdLoadCallback
import dagger.hilt.android.qualifiers.ApplicationContext
import kotlinx.coroutines.flow.MutableSharedFlow
import kotlinx.coroutines.flow.SharedFlow
import kotlinx.coroutines.flow.asSharedFlow
import javax.inject.Inject
import javax.inject.Singleton

@Singleton
class AdManager @Inject constructor(
    @ApplicationContext private val context: Context,
    private val billingManager: BillingManager
) {
    private val cadence = InterstitialCadence()
    private var interstitialAd: InterstitialAd? = null

    private val _showInterstitialEvents = MutableSharedFlow<Unit>(extraBufferCapacity = 1)
    val showInterstitialEvents: SharedFlow<Unit> = _showInterstitialEvents.asSharedFlow()

    init {
        MobileAds.initialize(context)
        loadInterstitial()
    }

    fun recordInteraction() {
        if (billingManager.isPremium.value) return
        val due = cadence.recordInteraction()
        if (due && interstitialAd != null) {
            _showInterstitialEvents.tryEmit(Unit)
        }
    }

    fun showInterstitial(activity: Activity) {
        val ad = interstitialAd ?: return
        ad.fullScreenContentCallback = object : FullScreenContentCallback() {
            override fun onAdDismissedFullScreenContent() {
                interstitialAd = null
                loadInterstitial()
            }

            override fun onAdFailedToShowFullScreenContent(adError: AdError) {
                AriaLogger.d("AdManager", "Interstitial failed to show: ${adError.message}")
                interstitialAd = null
                loadInterstitial()
            }
        }
        cadence.markShown()
        ad.show(activity)
    }

    private fun loadInterstitial() {
        InterstitialAd.load(
            context,
            BuildConfig.ADMOB_INTERSTITIAL_AD_UNIT_ID,
            AdRequest.Builder().build(),
            object : InterstitialAdLoadCallback() {
                override fun onAdLoaded(ad: InterstitialAd) {
                    interstitialAd = ad
                }

                override fun onAdFailedToLoad(error: LoadAdError) {
                    AriaLogger.d("AdManager", "Interstitial failed to load: ${error.message}")
                    interstitialAd = null
                }
            }
        )
    }
}
```

```kotlin
import androidx.compose.foundation.layout.fillMaxWidth
import androidx.compose.runtime.Composable
import androidx.compose.ui.Modifier
import androidx.compose.ui.platform.LocalContext

@Composable
fun BannerAdView(modifier: Modifier = Modifier) {
    val context = LocalContext.current
    AndroidView(
        modifier = modifier.fillMaxWidth(),
        factory = {
            AdView(context).apply {
                setAdSize(AdSize.BANNER)
                adUnitId = BuildConfig.ADMOB_BANNER_AD_UNIT_ID
                loadAd(AdRequest.Builder().build())
            }
        }
    )
}
```
